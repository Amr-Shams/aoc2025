
# Advent of Code 2025

## Setup

1. Install OCaml and Nushell.
2. Set your Advent of Code session:

```nu
$env.AOC_SESSION="your_session_cookie"
````
```nu
$env.AOC_YEAR = 2025
```


## Initialize Project

```nu
nu scripts/init.nu
```

## Run a Solution
this will run day 1 part 1
```bash
nu scripts/run.nu 1 1
```

* First argument: `1` or `2` for part 1 or part 2
* Second argument: path to the input file
