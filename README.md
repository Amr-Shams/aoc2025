
# Advent of Code 2025

## Setup

1. Install OCaml and Nushell.
2. Set your Advent of Code session:

```nu
$env.AOC_SESSION="your_session_cookie"
$env.AOC_YEAR = 2025
```


## Initialize Project

```nu
nu aoc.nu init --lang --days 
```

## Run a Solution
this will run day 1 part 1
```bash
nu aoc.nu run 1 1
```
## Help 

```bash 
nu aoc.nu --help
```
