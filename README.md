
# Advent of Code 2025

## Setup

1. Install OCaml and Nushell.
2. Set your Advent of Code session:

```nu
export AOC_SESSION="your_session_cookie"
````

## Initialize Project

```nu
nu scripts/init.nu
```

## Run a Solution

```bash
ocaml solutions/day01.ml 1 inputs/day01.txt
```

* First argument: `1` or `2` for part 1 or part 2
* Second argument: path to the input file
